# Next.js 윈도우 크기 추적 애플리케이션

이 프로젝트는 브라우저 창의 크기를 실시간으로 추적하는 커스텀 React 훅(`useWindowSize`)을 사용하는 간단한 Next.js 애플리케이션입니다. 브라우저 크기 변경에 따라 화면에 창의 너비와 높이가 업데이트됩니다.

## 주요 기능
- 브라우저 창 크기의 실시간 추적
- 재사용 가능한 커스텀 React 훅 구현
- Next.js 13+ App Router 구조 사용
